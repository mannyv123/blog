---
title: Object.fromEntries
date: 2023-04-09
tag: JS
---

TIL that there's a nifty methd on the `Object` type in JS: `fromEntries`. It's the 
counterpart of `entries` method that exists on the `Object`.

Here's an example for you: 

```js
const entries = Object.entries({ foo: "bar", baz: 42 });
console.log(entries);
// Expected output: [["foo", "bar"], ["baz", 42]]

const obj = Object.fromEntries(entries);
console.log(obj);
// Expected output: Object { foo: "bar", baz: 42 }
````
